The Nordea Codes app changes name to Nordea ID app

It’s the same app you know – just with a new name. You will see the new name when you update the app to the latest version (2.0). You can continue to use the app as before. As a new feature, the app lets you use a QR code to log in to the new Nordea Netbank and various third-party services. Read more at nordea.fi/IDapp

BankID QR code

Generate a one-time QR code to log in without typing your user ID.

  1. Click OK to generate the code.
  2. Open the BankID app on your phone or tablet.
  3. Tap the Scan QR code button. This opens a camera view for scanning the code.
  4. Point the camera at the QR code. Make sure to fit the entire code into the scanner window.
  5. Enter your PIN code or confirm with biometric authentication.

Login identifier

The identifier helps you confirm the right login. Check that the identifier shown by the ID app matches the one on this page.

If the identifiers do not match, ignore the request – do not approve it.
